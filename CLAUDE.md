# 스마트스토어 자동화 (smartstore_auto)

Railway 배포 서비스: https://smartstore-auto-production.up.railway.app

## 역할
네이버 스마트스토어 상품 자동 소싱 및 등록 시스템.

## 주요 파일
- `main.py` — 진입점
- `employees.py` — 자동화 작업 모듈

## 실행 환경
- Python, Railway 배포
- 환경변수: NAVER_CLIENT_ID, NAVER_CLIENT_SECRET, SMARTSTORE_NAVER_ID, SMARTSTORE_NAVER_PW

## 주의사항
- Windows 환경. find/grep/xargs 대신 Python pathlib/glob 사용
- 수정 후 Railway 자동 배포됨

## 블루오션 상품 선정 로직

블루오션 상품 선정 시 **오늘 날짜를 먼저 확인**하고 아래 기준을 자동 적용한다.

### 1. 계절 기준 (현재 월 기준 자동 적용)
| 계절 | 월 | 추천 카테고리 키워드 |
|------|-----|----------------------|
| 봄   | 3~5월 | 꽃샘추위, 봄나들이, 청소, 수납, 원예 |
| 여름 | 6~8월 | 냉감, 캠핑, 수납, 주방, 물놀이, 선풍기, 보냉 |
| 가을 | 9~11월 | 단풍, 캠핑, 수납, 홈인테리어, 건강 |

### 3. 절대 제외 조건
- **현재 계절과 맞지 않는 상품** (예: 여름에 발열조끼/핫팩, 겨울에 냉감/물놀이)
- **브랜드명 포함** (삼성, LG, 나이키, 스탠들러 등)
- **캐릭터/애니메이션 관련** (산리오, 디즈니, 포켓몬, 아기상어 등)
- **이미 영상 제작한 상품**:
  - 에펠탑 미니어처 프라모델 조립키트
  - 이중유리 커피 추출기 800ml
  - 산리오 포토 키링 DIY 세트
  - 귀마개 샤워캡
  - 풍선 가랜드
  - 실리콘 주방장갑
  - 시스맥스 서랍책꽂이

### 4. 선정 절차
1. `date` 명령으로 오늘 날짜 확인 → 현재 월 파악
2. 현재 계절 + 이벤트 키워드 도출
3. `/list-products` 조회 → 가격 10,000~50,000원 필터
4. 계절·이벤트 키워드와 일치하는 상품 우선 선택
5. 도매꾹 API(`getItemList`)로 원본 이미지 URL 확보
6. 최종 4개 선정 → 상품명/가격/이미지URL/스마트스토어URL/특징3가지 정리
